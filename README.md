# NineMusic
iOS 9 Now Playing Lockscreen Interface for iOS 13 & 14.
 
## Compatibility
- Supports iOS 13.0 to 14.2

## Installation
1. Add 'https://minh-ton.github.io/repo' to your package manager
2. Search for NineMusic and install the tweak.
